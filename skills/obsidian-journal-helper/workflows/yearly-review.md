# Yearly Review Workflow

Use this workflow to review a year from monthly notes, weekly notes, and an optional existing yearly note.

Before reviewing, apply `guides/journal-principles.md`.

## Inputs

Read:

- available monthly notes from the requested year;
- weekly notes when monthly notes are missing or insufficient;
- the yearly note, if it already exists.

## Section Cleanup Rules

- If a section is empty, ask whether it should be removed.
- If a section contains user content, remove the instructional comment directly under that section heading.
- Do not remove a section only because it has no content unless the user confirms.

## Steps

1. Identify the year.
   Completion criterion: the `YYYY` identifier is known.

2. Find available sources.
   Completion criterion: monthly notes, fallback weekly notes, and existing yearly note are accounted for.

3. Read available sources.
   Completion criterion: the review considers the available year without treating missing months as facts.

4. Extract yearly facts.
   Completion criterion: major events, wins, difficulties, lessons, and changes trace to source notes.

5. Identify long-range patterns.
   Completion criterion: identity shifts, values, tensions, and direction are labeled as interpretations unless directly stated.

6. Prepare the review.
   Completion criterion: the output contains every heading from the response format.

7. Check the result against `guides/journal-principles.md`.
   Completion criterion: the review is nonjudgmental and does not overstate the evidence.

## Response Format

```markdown
## Year Motto

## Key Events

## Biggest Wins

## Hardest Moments

## Life Areas

### Health

```
