# Weekly Review Workflow

Use this workflow to review a week from daily notes and an optional existing weekly note.

Before reviewing, apply `guides/journal-principles.md`.

## Inputs

Read:

- available daily notes from the requested week;
- the weekly note, if it already exists;
- the linked monthly note only when needed for broader context.

## Section Cleanup Rules

- If a section is empty, ask whether it should be removed.
- If a section contains user content, remove the instructional comment directly under that section heading.
- Do not remove a section only because it has no content unless the user confirms.

## Steps

1. Identify the week.
   Completion criterion: the `YYYY-Www` identifier and date range are known.

2. Find available sources.
   Completion criterion: available daily notes, the existing weekly note, and optional monthly context are accounted for.

3. Read available entries.
   Completion criterion: each available daily note is considered, and missing days are not treated as facts.

4. Extract weekly facts.
   Completion criterion: events, decisions, energy signals, and lessons trace to source notes.

5. Identify patterns.
   Completion criterion: repeated themes, tensions, and hypotheses are labeled as interpretations.

6. Prepare the review.
   Completion criterion: the output contains every heading from the response format.

7. Check the result against `guides/journal-principles.md`.
   Completion criterion: the review states limitations when the source base is sparse.

## Response Format

```markdown
## Week Summary

## Wins

## Difficulties

## Lessons

## Health / Energy

## Work / Learning

## Relationships

## Money

## Next Week

### Priorities

## Reflection Question
```
