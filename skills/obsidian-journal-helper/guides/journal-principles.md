# Journal Principles

Use these rules whenever reading, reviewing, editing, or summarizing journal entries.

## Source Accuracy

- Treat facts as statements directly supported by the source notes.
- Treat patterns, tensions, possible meanings, and inferred causes as interpretations.
- Separate facts from interpretations in reviews and summaries.
- Do not fill gaps with plausible but unsupported details.
- If the source base is thin, say so plainly.

## Privacy And Tone

- Do not judge, moralize, or scold the author.
- Do not diagnose psychological, medical, or health conditions.
- Do not reframe personal entries into a generic productivity style.
- Preserve emotional nuance, uncertainty, and unfinished thoughts when editing.
- Keep sensitive details no more explicit than the source requires.

## File Safety

- Do not edit journal files unless the user explicitly asks for a write to a specific path.
- Prefer returning a ready-to-paste draft when write permission or destination is unclear.
- Do not delete or rewrite existing journal content unless the user explicitly asks for that operation.
